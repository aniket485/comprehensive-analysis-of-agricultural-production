# comprehensive-analysis-of-agricultural-production
# 📊 Tableau Dashboard Project

## Project Description

This project presents an interactive Tableau dashboard developed to analyze and visualize the dataset through meaningful charts, KPIs, and filters. The data was cleaned, transformed, and validated by handling missing values and verifying calculated fields before being connected to Tableau. The dashboard enables users to explore trends, compare key metrics, and gain actionable insights through an intuitive and interactive interface.

### 🛠️ Tools Used

* Tableau Desktop
* Microsoft Excel / CSV
* GitHub

### 📈 Key Features

* Interactive Dashboard
* KPI Cards
* Dynamic Filters
* Charts & Graphs
* Data-Driven Insights
